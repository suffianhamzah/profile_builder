# Atlas Project

## Goal
Deliver a simple working implementation of the application described in `requirements.md` within 2 hour constraint

## Priorities
1. Correct core behavior
2. Clear and maintanable code
3. Focused tests
4. Explicit assumptions and tradeoffs

## Constraints
1. Dont add additional dependencies without a clear need
2. Avoid abstractions that are not required
3. Preserve public interfaces unless explicitly asked
4. Run tests after a minimum change

## Coding Principles
1. Simple and clear code
2. Separation of responsibilities
3. Clear domains

## Repository
 - `src/` : implementation
 - `tests/`: tests
 - `docs/decisions.md`: assumptions and tradeoffs
